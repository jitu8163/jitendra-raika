
# Jitendra Raika — Bold & Colorful Portfolio

A single-page portfolio website with vibrant gradients, bright accent colors, and engaging animations showcasing Jitendra's AI/ML expertise.

## Sections

### 1. Hero Section
- Full-screen hero with a bold gradient background (purple → blue → teal)
- Name "JITENDRA RAIKA" in large, striking typography
- Tagline: "AI/ML Engineer | Data Scientist"
- Location, email, LinkedIn, and portfolio links as icon buttons
- Animated call-to-action buttons: "View Projects" and "Contact Me"

### 2. About / Summary
- Eye-catching card with the professional summary
- Colorful accent border/decoration

### 3. Skills
- Categorized skill badges with vibrant color-coded groups (e.g., blue for Programming, green for ML/AI, orange for Tools)
- Categories: Programming, Databases, Machine Learning, Data Analysis, Visualization, AI Architectures, Frameworks, Tools, Soft Skills

### 4. Experience Timeline
- Vertical timeline with colorful markers for each role
- Three positions: Telepathy Infotech (AI/ML Engineer Intern), Maxgen Technologies (Data Scientist Intern), TechieNest (AI-ML Intern)
- Key achievements as bullet points under each role

### 5. Projects Showcase
- Grid of project cards with gradient borders and hover effects
- 5 projects: RAG Knowledge Assistant, College Policy Chatbot, Potato Disease Detection, JARVIS Virtual Assistant, Customer Churn Prediction
- Each card shows project name, description, and tech used

### 6. Education
- Colorful cards showing B.Tech (SGPA 8.2), Higher Secondary (95.80%), Senior Secondary (87.50%)

### 7. Certifications
- Horizontal scrollable badges or a grid of certification cards with icons

### 8. Contact / Footer
- Contact section with email and LinkedIn links
- Languages: English (C1), Hindi (C2)

## Design Details
- Bold gradient backgrounds transitioning between sections
- Bright accent colors (purple, teal, orange, pink)
- Smooth scroll navigation with a sticky top navbar
- Subtle entrance animations as sections scroll into view
- Fully responsive (mobile-friendly)
- No backend needed — all data is static
